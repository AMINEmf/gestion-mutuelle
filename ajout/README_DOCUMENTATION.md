## 📋 Résumé court (30 sec)

**Le panel de suggestions des postes** est composé de:

1. **PosteSuggestionsDrawer.jsx** - Compose qui affiche (stateless)
2. **PositionsGrades.jsx** - Parent qui gère état et API (smart)
3. **API /postes/{id}/suggestions** - Suggère employés avec scores IA
4. **CSS .cnss-*** - Styles partagés en design system
5. **Backend Laravel** - Calcule compatibilité scores

Le flux: User → Click → State change → API call → Data → Render → Drawer display
